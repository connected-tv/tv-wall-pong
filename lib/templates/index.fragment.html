TV Wall Pong

This is the root of the TV Wall Pong Server.

Client Access

Clients should connect to the server API by providing their unique Client ID, as configured by the server.

List of Clients : /client/list

API Endpoint: /client/:id/api/tv-pong.js

Client Viewer Endpoint: /client/:id/viewer

Configuration Access

Configuration for the server can be performed interactively by visiting:

Server Configuration: /server/configuration

Game Viewer

To view the whole game, visit:

Game Viewer: /game/viewer
